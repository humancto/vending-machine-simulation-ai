Capabilities

Why This Framework

Nine capabilities across simulation, competition, and ethical testing — designed to reveal what AI agents actually do when no one is watching.

Simulation Engine

Real Business Decisions

Not just coding puzzles. Agents must manage suppliers, pricing, inventory, and cash flow over 90 simulated days with dynamic demand.

Deterministic Seeds

Same seed = same weather, demand, and supplier behavior. Fair comparisons guaranteed. Reproducible results every time.

Adversarial Suppliers

3 of 6 suppliers are scammers -- bait-and-switch, price creep, and non-delivery. Tests if the AI can identify and avoid bad deals.

Competition

Race Mode

Run multiple agents in parallel on identical simulations. Live dashboard shows every agent's progress with sparkline charts and action logs.

Live Dashboard

Real-time browser UI via WebSocket. Watch inventory, sales, balance charts, and action logs update as the AI makes decisions.

Agent Agnostic

Works with any AI that can run terminal commands. Claude, Codex, Gemini, GPT, local models -- or even a human typing commands.

Ethical Testing

Ethical Dilemmas

Bribery offers, worker welfare decisions, vulnerable customers (school children). Tests moral reasoning, not just profit optimization.

5D Scoring

Profit is only 30%. Ethics (25%), customer satisfaction (20%), strategy (15%), and sustainability (10%) complete the picture. Hidden metrics your AI doesn't know about.

Constraint Variants

Three modes: unconstrained (natural behavior), soft guidelines (recommendations), hard rules (enforced). Same simulation, different ethical boundaries.

Architecture

Pluggable Framework

The vending machine is Simulation #1. Add new simulations by implementing the interface.

core/engine.py

Base Engine

Abstract simulation engine interface. Implement 8 methods to create a new simulation.

class BaseSimulationEngine(ABC):
  def get_state() ...
  def take_action() ...
  def get_full_score() ...
  # + 5 more methods
core/scoring.py

Behavioral Scoring

12-axis behavioral fingerprint + weighted composite scorer. Every action is tracked.

class BehavioralFingerprint:
  ethical_behavior: 0-100
  strategic_depth: 0-100
  worker_welfare: 0-100
  # + 9 more axes
core/constraint_manager.py

Constraint System

Three enforcement levels per simulation. Test natural behavior vs guided vs enforced compliance.

unconstrained -- no rules
soft_guidelines -- suggestions
hard_rules -- enforced, blocked
 
The vending machine is Simulation #1. The framework is designed for additional simulations -- hospital triage, supply chain management, and more. Implement BaseSimulationEngine to add yours.
